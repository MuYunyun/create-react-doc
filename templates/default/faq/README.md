这里可以问一些常见问题。

1. 是否支持 Markdown 中写代码？

> 支持在 Markdown 中写HTML、CSS。

2. 文档是否支持目录层级？

> 支持菜单无线层级，在文件夹中建立文件夹，相对应建立 `README.md`

3. 新增文件不能热加载，监听文件变化？

> 新增一个 `Markdown` 文件是支持热加载的，需要如下步骤：
> - 添加完成之后必须 `Markdown` 文件不能为空。
> - 如果只是添加注释配置或者空格，一样视为空。
> - 在其它已加载的 `Markdown` 文件中保存一下。
