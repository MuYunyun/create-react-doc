## Front-matter

Front-matter 是文件最上方包裹在 `<!--` 与 `-->` 之间的区域，用于指定个别文件的变量，举例来说：

```md
<!--
abbrlink: 290a4219
-->
```

以下是预先定义的参数，您可在模板中使用这些参数值并加以利用。

| 参数     | 描述                                                            | 默认值 |
| :------- | :-------------------------------------------------------------- | :----- |
| abbrlink | 短链。用于指定页面路由展示为指定短链，使用短链有助于 SEO 搜索。 |        |
